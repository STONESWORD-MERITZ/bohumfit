# BOHUMFIT-107-q2-clinical-review-likelihood

## 목적
건강체 Q2(1년 이내 진단) 카드의 추가검사·재검사 확인 안내를 개선한다.

## 배경
- Q2 항목은 1년 이내 확정진단이면 표시된다.
- 실제 추가검사·재검사를 받았는지와 별개로, 병원에서 추가검사·재검사 필요 소견이나 권유가 있었는지 사용자가 확인해야 한다.
- 기존 화면 문구는 "자동 의심 소견 없음 - 원자료 기준 추가검사·재검사 여부 확인"이라 사용자가 무엇을 확인해야 하는지 직관성이 약하다.

## Scope
- `src/pages/Disclosure.tsx`

## 작업
- Q2 임상 확인 배지를 `가능성 높음`, `가능성 낮음`, `가능성 미확인`으로 구분해 표시한다.
- `q2_suspicion` 또는 `additional_test_reason`에 포함된 기존 판단 문구를 보존하되, 화면에서는 가능성 수준과 근거를 읽기 쉽게 정리한다.
- 자동 판단이 없는 항목은 실제 검사 시행 여부가 아니라 "의사의 추가검사·재검사 필요 소견/권유 여부"를 확인하라는 안내를 표시한다.
- 백엔드 판정 로직, AI 호출 조건, 분석 결과 구조는 변경하지 않는다.

## 검증
- `npx tsc -p tsconfig.app.json --noEmit`
- `npx tsc -p tsconfig.node.json --noEmit`
- `npm run lint`
- `npm test`
- `npm run build`

## 완료 조건
- Q2 카드에서 추가검사·재검사 가능성 수준이 보인다.
- 자동 판단이 없는 경우도 사용자가 확인할 질문이 명확히 보인다.
- 검증 통과 후 범위 파일만 커밋·푸시한다.
