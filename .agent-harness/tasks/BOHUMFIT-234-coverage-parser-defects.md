## S3 — 사양 결정지 (수정 금지·보고만)

1. **⑧ 납입완료 계약**: ★결정적 실측 — A의 KB 원본 헤더 월납합 2,282,564 = 전체 합 2,835,744 − **납입완료 4계약 합 553,180**(2003·2004·2007년 가입 10~15년납). 즉 KB 자체가 납입완료 계약을 월납 합계에서 제외한다. 옵션: (a) KB 정합 — 가입일+납입년수 경과 계약을 월납 합산 제외+"납입완료" 배지 표기(총납입에는 포함) ← KB 산식과 일치·권장 (b) 현행 유지(전체 합산) + 헤더 값 병기 (c) KB 헤더 값을 직접 파싱해 표시. (a)는 날짜 기준 판정이라 테스트 시 기준일 주입 필요.
2. **B 요청 — 계피관계·납만기 컬럼**: 데이터는 이미 파싱됨(remark 계피동일/상이, pay_years·maturity) — 원본에 존재 확인. 리포트/화면 컬럼 추가는 프런트 표시 사양(저위험 후속).
3. **③ 계약 번호 렌더 정렬**: 다중행 셀에서 번호가 세로 중앙 정렬돼 옆/밑 혼동 — 번호·처리 열 상단 정렬(또는 행 병합 표시)로 통일하는 프런트/리포트 템플릿 후속.
4. (기록) "5대골절수술비"는 여전히 N대수술비로 포섭됨(문건주 50만) — 골절 그룹 분리 여부는 후속 판단.
