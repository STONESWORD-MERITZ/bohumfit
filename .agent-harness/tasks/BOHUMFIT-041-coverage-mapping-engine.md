   상해·질병후유장해/암진단금/유사암/표적항암/차세대암/암수술/뇌혈관·뇌졸중·뇌출혈/뇌혈관수술/
   허혈·급성심근경색/심혈관수술/일반종수술1~5종/상해·질병수술/질병·상해입원/응급실내원/
   골절·화상진단/운전자특약·자부상·가일배·상해실손·질병실손(Y/N형)/보험료
2. **보장명→카테고리 매핑 사전(JSON)**: 원천자료 보장명 기준 시드
   (질병사망, 상해사망, 암진단, 고액암진단→암진단금, 소액암·특정암→유사암,
    표적항암약물치료비→표적항암, 뇌졸중진단→뇌졸중(중기), 급성심근경색진단→급성심근경색(말기),
    질병·상해종수술→일반종수술, 입원일당류→질병/상해입원, 골절·중대골절→골절진단비 등)
   매핑 불가 보장명은 unmapped 로 반환(드롭 금지 — 수동 배정용)
3. **변환 함수**: 계약별 담보 배열 → 비분표 열 데이터
   - 사망 분해 룰: 질병사망=상해사망→일반사망 / 질병<상해→일반=질병, 재해=차액 /
     질병사망 없음→상해사망 행 유지 (검증례: 질병1억+상해3억→일반1억+재해2억)
   - 종수술 자동셋팅 기준표 {100:[5,15,25,50,100], 500:[10,30,50,100,500],
     600:[20,60,100,200,600], 1000:[20,50,100,500,1000], 3000:[50,100,200,1000,3000]} (만원)
     구간 사이 금액은 인접 구간 선형 평균(보간)으로 1~4종 제안, 5종=가입금액.
     suggested 플래그 포함(후속 UI에서 전 칸 수정 가능해야 함)
   - Y/N형 카테고리는 금액 아닌 존재 여부
   - 합계 계산(전/후 공용 순수 함수)
4. **단위테스트**: 원천자료 샘플 기반 케이스(교보/신한/흥국/삼성/AIA) + 사망분해 3분기 +
   종수술 정확구간/보간구간 + unmapped 처리
5. **컨설팅 후 계산 모델(043 대비 — 데이터 모델에 포함)**
   - 계약 상태: 유지 / 해지 (계약 단위)
   - 유지 계약의 담보별 감액 override: 담보 단위 조정 가입금액 + 보험료 수기 조정값 구조
   - 후 비분표 = 유지 계약(override 반영) + 신규 제안 계약 배열
   - 전/후 동일한 순수 함수로 계산(입력만 다르게) — 후 전용 로직 금지
6. (선택) `parseSourceRows(rows) → 계약[]` 시그니처만 정의(구현은 042 — SheetJS 결과 연결 자리)
