# BOHUMFIT-106-surgery-filter-audit
## 목표
수술 탐지 로직 전수조사 및 오분류 패턴 일괄 보정
## 배경
- BOHUMFIT-104에서 부목·캐스트 오분류 수정 완료
- 새 오분류: "한냉치료(냉동치료)"가 수술로 오분류(2024-06-07 본정형외과의원, S93, 세부 "한냉치료(냉동치료)")
- 한냉치료는 이학요법(물리치료)이며 수술 아님 → 전수조사로 유사 패턴 일괄 보정
## 전수조사 항목
1. surgery_exclusions.py 현재 _NON_SURGERY_ACTION_KEYWORDS 전체 확인
2. 이학요법/물리치료 계열 누락 확인:
   한냉치료/냉동치료/냉찜질, 표층열치료/심층열치료, 간섭파전류치료/ICT, 적외선/초음파치료,
   재활치료/재활저출력레이저, 도수치료/운동치료, 견인치료/전기자극치료, 온열치료/초단파치료
3. 기타 비수술 처치: 좌욕, 단순처치, 산소흡입, 도뇨, 냉찜질/온찜질, 드레싱/창상처치
4. 세부진료 "이학요법료/이학요법" 카테고리 행이 수술로 잡히는지 → 카테고리 예외 검토
5. 강수술 신호와 충돌 없는지 검증(진짜 수술 누락 0)
## 수정 지침
