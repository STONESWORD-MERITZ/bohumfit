# BOHUMFIT-101-layout-subscription-fix
## 목표
1. 전체 페이지 상단 여백 통일
2. 구독 페이지 카드 버튼 밸런스 + 오픈 이벤트 기간 명시
## 배경
- 홈·고지의무분석·실손계산은 헤더 바로 아래부터 콘텐츠 시작(체감)
- 자료받기·보장비교분석·보험사링크·요금제 등은 콘텐츠가 안쪽 padding에서 시작(체감)
- 구독 페이지: 베이직 카드에 이벤트 배지가 있어 프로 카드 버튼보다 아래 위치
- 오픈 이벤트 기간이 "첫 3개월"로만 표기, 실제 날짜 미명시
## 작업 범위
- src/components/Layout.tsx 또는 페이지 공통 wrapper (상단 여백 통일)
- src/pages/ 각 페이지 상단 pt/mt 클래스
- src/pages/Subscription.tsx (구독 페이지)
## 수정 지침
### 1. 페이지 상단 여백 통일
- 기준: 헤더 높이 아래 pt-8(32px) 또는 pt-10(40px) 통일
- 홈(Landing)처럼 히어로 섹션이 배경색을 꽉 채우는 페이지는 예외 허용
### 2. 구독 카드 버튼 밸런스
- 베이직·프로 카드 동일 높이, 버튼 mt-auto 하단 고정
### 3. 오픈 이벤트 기간 명시
- "오픈 이벤트 · 첫 3개월" → "오픈 이벤트 · ~2026년 9월 30일"
- "이벤트 후 월 14,900원 · 매월 30회" 현행 유지
- 추가: 작은 텍스트 "2026년 9월 30일까지 적용"
