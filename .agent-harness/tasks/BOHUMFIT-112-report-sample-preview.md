# BOHUMFIT-112: 고지의무 리포트 샘플 미리보기 (비로그인 구독 유도)
## 목표
비로그인 사용자가 고지의무 분석 리포트가 어떻게 생겼는지 샘플로 미리 볼 수 있게 하여 구독 유도.
## 구현 방향
### 샘플 데이터
- 실제 분석 결과 구조와 동일한 mock 데이터를 프런트에 하드코딩
- 민감정보 없는 가상의 질병명/투약일수/수술 여부 사용
### 진입 경로 (코드 구조에 맞는 방식 선택)
A. /disclosure/sample 신규 라우트 / B. Disclosure.tsx에 sample 모드 prop
### 노출 위치
- 홈(Home.tsx) 또는 WhyDisclosure.tsx에 "리포트 샘플 미리보기" 버튼
- 샘플 화면 상단에 "이것은 샘플입니다. 실제 분석을 원하시면 구독해주세요" 배너 + 구독 페이지 이동 버튼
## 검증
- npx tsc app/node / npm run lint / npm test
## 완료 조건
- 비로그인 상태에서 샘플 리포트 접근 가능
- 샘플 배너 및 구독 유도 버튼 표시
- 실제 분석 데이터와 혼동 없음
