# BOHUMFIT-128 추가검사/재검사 UI 개편
## 배경
Q2(1년) 추가검사·재검사 대상이 고지 복사 텍스트에 포함되고 문구가 시스템 한계처럼 보여 혼란.
## 스펙
### 백엔드(result_builder.py)
- 추가검사/재검사 대상 질병에 `exam_check_only: true` 플래그. 고지 복사 텍스트 생성 시 해당 항목 제외.
### 프런트(src/pages/Disclosure.tsx)
- Q2를 [A]일반 고지(기존) + [B]설계사 참고용(별도 박스·연한색·복사버튼 없음)로 분리.
  [B] 제목 "설계사 확인 필요 항목", 부제 "고지 대상은 아니나 아래 질병에 대해 의사의 추가검사·재검사 소견 또는 권유가 있었는지 고객에게 직접 확인해 주세요."
### 문구 변경
- 신규: "검사 시행 여부와 관계없이, 의사로부터 추가검사나 재검사가 필요하다는 소견 또는 권유를 받으셨는지 고객에게 직접 확인해 주세요."
## 작업
- Step1 분석: result_builder 추가검사 판별·복사텍스트 생성 위치, Disclosure.tsx Q2 렌더·복사버튼.
- Step2 백엔드: exam_check_only 플래그 + 복사텍스트 제외.
- Step3 프런트: Q2 [A]/[B] 분리·스타일·문구·[B] 복사버튼 미표시.
- Step4 검증: pytest·tsc·build + 신규(a exam_check_only 복사 미포함 / b 일반 복사 포함).
## 수정 금지: Q1/Q3/Q4 섹션·filters.py(고지 판정) 수정 금지.
