# BOHUMFIT-126 상해 S코드 초진/재진 기반 그룹 분리
## 배경
S코드(상해)는 동일 코드라도 초진이면 새 상해 → 별개 그룹. 현재 동일 S코드를 날짜 누적으로 한 그룹으로 묶어 통원/입원일수 오집계.
## 스펙
- 세부진료 `진료내역`에 "초진" → 새 상해 그룹 시작 / "재진" → 직전 그룹 연속
- S00~S99 한정. 초진 여러 번이면 각각 별개 그룹. 초진/재진 정보 없으면 기존 로직 유지(하위호환).
## 작업
- Step1 분석: pdf_parser 진료내역 초진/재진 처리, aggregator S코드 그룹키·초진/재진 필드 → handoff.
- Step2 수정: pdf_parser `is_first_visit:bool` 추출; aggregator S코드 그룹키에 초진일 포함(S63|20180620). 재진=최근 초진 그룹 귀속. 초진정보 없으면 기존.
- Step3 검증: pytest 전체 + 신규(a 초진2→그룹2 / b 초진1+재진3→그룹1·통원4 / c 비S코드 초진2→그룹1 유지).
## 수정 금지: 명시 외 파일·프런트.
