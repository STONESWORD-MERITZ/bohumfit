# BOHUMFIT-084: 병력 자료 다운로드 가이드 전면 개편 (심평원 + 건강보험공단)
## 목적
설계사가 고객 병력 PDF를 정확히 수집하도록 DownloadGuide 페이지를 심평원(HIRA)·국민건강보험공단(NHIS) 두 기관 기준으로 전면 재작성한다.
- 왜 두 기관 모두?: 심평원은 최근 5년 상세 진료·처방, 공단은 최대 10년 요양급여 이력 → 장기 병력 누락 방지.
## 대상 파일
- src/pages/DownloadGuide.tsx (전면 수정). 필요 시 하위 컴포넌트 분리 가능.
## 요구사항
### 1. 페이지 구조 (섹션/탭 2개)
- 섹션 A: 건강보험심사평가원(HIRA) — "내 진료정보 열람"
- 섹션 B: 국민건강보험공단(NHIS) — "건강보험 요양급여내역 조회"
- 상단: 두 기관에서 모두 받아야 하는 이유 1~2줄 + 하단 체크리스트로 이동 버튼
### 2. 섹션 A — 심평원(HIRA), 링크 https://www.hira.or.kr/main.do
1) 로그인 후 [조회·신청 > 내 진료정보 열람] 진입
2) '고유식별정보 처리 고지' 화면 → ['내 진료정보 열람' 조회하기]
3) 로그인 방식 선택(간편인증/공동인증서/금융인증서/모바일 신분증)
4) 진료정보 조회 설정:
   - 보험종별: [건강보험, 의료급여, 보훈] 선택
   - 상병항목 표시: 체크
   - 민감상병 표시: 체크 (★ 고지의무 분석 필수)
   - 전체 병·의원&약국: 체크
   - 대상기간: [5년]
5) 다음 3개 탭에서 각각 [전체 다운로드]: 기본진료내역 / 세부진료정보 / 처방조제정보
6) 보험종별을 [자동차보험]으로 변경 후 2개 탭 [전체 다운로드]: 기본진료내역 / 세부진료정보
→ 심평원 합계: 5개 파일
### 3. 섹션 B — 건강보험공단(NHIS), 링크 https://www.nhis.or.kr/nhis/index.do
1) 로그인 후 상단 [통합검색]에서 "요양급여" 검색
2) [건강보험 요양급여내역 조회] → [조회하기]
3) 1단계 유의사항 → 2단계 사전면담 → 3단계 내역조회
4) 진료개시일 기간 설정:
   - ★ 조회 구간 최대 1년 → 1년 단위로 끊어 조회(예: 2021.01.01~2021.12.31, 2022.01.01~2022.12.31 …)
   - 최대 10년 치까지 가능
5) 특수상병 포함 여부: 체크
6) [조회] → [프린트발급]으로 PDF 저장(연도별 1개씩)
→ 공단 합계: 5년 기준 5개(1년 1개), 최대 10개
### 4. 최종 체크리스트 컴포넌트(화면 내 useState로 체크 상태 관리)
- (심평원) 건강보험 기본진료내역 / 세부진료정보 / 처방조제정보
- (심평원) 자동차보험 기본진료내역 / 세부진료정보
- (공단) 요양급여내역 1년차~5년차(필요 시 최대 10년차)
### 5. 스크린샷
- 각 단계에 예시 스크린샷 자리를 플레이스홀더 박스로 둔다(예: /images/guide/hira-step4.png).
- ★ 개인정보(이름·주민번호·진료내역)가 포함된 실제 캡처 사용 금지. 추후 마스킹 예시로 교체.
### 6. 스타일/타이포
- 082 한국어 모바일 타이포 규칙 유지(word-break: keep-all, --font-ko 등). 기존 토큰/컴포넌트 재사용. 모바일 우선.
