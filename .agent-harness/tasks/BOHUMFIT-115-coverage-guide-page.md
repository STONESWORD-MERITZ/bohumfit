# BOHUMFIT-115: 보장분석서 PDF 다운로드 가이드 페이지

## 목표
보험사별(한화손보/KB손보/DB손보) 보장분석서 PDF를 받는 방법을 안내하는 가이드 페이지를 신규 추가한다.

## 진입 경로
1. `/coverage-compare` 보장 비교분석 Step1 업로드 영역 상단
   - "PDF 받는 방법을 모르신다면 -> 보험사별 가이드 보기" 링크
2. `/coverage-guide` 신규 공개 라우트
3. 자료 받기 페이지
   - "보장분석서 받기" 섹션 또는 링크 추가

## 신규 파일
### `src/pages/CoverageGuide.tsx`
탭 구조로 3개 보험사 안내:
- 한화손보
- KB손보
- DB손보

공통 구성:
- 상단: "보장 비교분석에 필요한 PDF를 받는 방법을 안내합니다"
- 보험사 탭
- 단계별 텍스트 안내
- 주의 문구 강조
- 하단: `/coverage-compare` 바로가기 버튼

## 한화손보 안내
1. 한화 GA 포탈(hanwhaga.com) 로그인 -> 고객등록 클릭
2. 가입설계동의 클릭
3. 문자동의(LMS) 탭 -> 고객명/생년월일/휴대폰 입력 -> 인증요청
4. 주민등록번호 입력 -> 실명인증
5. 주소 입력 -> 저장
6. 메인화면 -> 실손/정액조회 -> 주민번호 탭 -> 고객이름 클릭
7. 화면 우측하단 [정액담보상세출력] 클릭
   - 주의: 일반 [출력] 버튼 사용 금지. 정액담보 미포함.
8. 프린터 -> Microsoft Print to PDF 또는 PDF24 선택 -> 저장
   - 주의: Microsoft Print to PDF 저장 시 이미지 PDF가 될 수 있음. 파싱 오류 발생 시 PDF24 프로그램으로 재출력.

## KB손보 안내
1. KB손보 MYMANAGER 영업포탈 로그인 -> 고객등록/수정 클릭
2. 동의서입력 클릭
3. 고객 주민번호/고객명/통신사/휴대폰번호 입력 -> 인증요청
4. 인증번호 입력 -> 등록 -> 닫기
5. 고객 휴대폰번호/주소/직업 입력 -> 저장
6. 메인화면 -> 보장분석 클릭
7. 가설동의순 클릭 -> 등록고객의 신규분석 클릭
8. 출력/발송 아이콘 클릭
9. 전체 선택/해제 클릭 -> PDF저장 클릭

## DB손보 안내
1. DB손해보험 영업포탈 로그인 -> 고객등록 클릭
2. 설계동의 클릭
3. 고객정보(고객명/생년월일/휴대폰) 입력 -> 발송
4. 인증번호 입력 -> 인증확인 -> 확인
5. 고객 상세정보(주민번호/주소/직업/운행차량) 입력 -> 저장
6. 메인화면 -> 스마트보장분석 클릭
7. 등록한 고객명 입력 -> 검색 -> 고객 더블클릭 -> 보장분석 클릭
8. 분석하기 클릭 -> 신용정보원 가입정보 조회 확인
9. 출력 클릭
10. 계약분석 + 세부가입내용 체크(필수) -> 출력하기
11. 환경설정 -> 페이지 품질 텍스트 방식 선택 -> 확인
12. 저장 클릭 -> PDF 저장
