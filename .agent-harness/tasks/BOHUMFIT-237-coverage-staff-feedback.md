## P2 — 조사·결정지 (동작 변경 0)

### D. 후유장해 3%이상 — 표기 전수(5개 실 PDF 채집, 원문 증거 태스크 이력 로그)

- 실측 표기 유형: ① 범위형 "상해 후유장해(3~100%)"(C p9 1억·H p10 1억), "(20~100%)"(D) —
  **"3%이상" 담보는 범위형 (3~100%)로 실재**. ② 임계형 "80%이상"·"50%이상"·"고도"(=80%이상 분류).
  ③ KB 분류 컬럼: 상해후유장해/상해80%이상후유장해/질병50%이상후유장해/교통상해후유장해 등.
- "80%미만" **담보명 표기는 detail 5개 PDF에서 0건** — 80%미만은 KB 기준담보(매트릭스 행)의
  명칭이지 상세 담보명이 아님.
- 판정 자료: (3~100%) 범위 담보와 "80%미만" 기준담보는 보장 범위가 다른 별개 개념 —
  80%미만 행을 "3%이상"으로 재라벨하면 **사실 왜곡**. 대안: 상세의 범위형 담보를 별도 라벨
  (예: "후유장해(3~100%)")로 EXTRA 가시화하는 안이 정직(2차 결정 항목).

### E. 기타 표시 우선순위 실측

- 현행 기타(및 EXTRA 파생 그룹) 노출 순서 = `raw["extra"]` dict 삽입 순 = **상세 페이지 등장 순**
  (파일마다 다름 — 고정 우선순위 없음). 화면 ④/⑤·엑셀·PDF 모두 이 순서를 그대로 렌더.
- 화상진단비 vs 상급/종합병원 일당 간 고정 우선순위를 원하면 EXTRA_PATTERNS 순서와 무관하게
  **표시 정렬 규칙(라벨 우선순위 표)** 신설이 필요 — 2차 결정 항목(원하는 순서 확정 필요).

### F. 종수술비 종별 분해 실측

- 원문 구조 2형: ① **종별 분리형**(D 케이스) — "질병1~5종수술비Ⅱ(...)_1종~_5종" 5행 각각 금액
  (10/20/50/300/500만), 상해도 동일 5행 — **원문에 종별 금액 실재 → 분해 표시 구현 가능**.
  ② **단일 합산형**(A 라이나) — "질병1~5종수술(수술당1회한)(5종수술) 1,000만" 1행뿐(종별 없음).
- 추정표 주입 요건(②형용): 스키마 `{보험사?, 상품패턴?, 종별비율: {1종:…5종:…}}` +
  표시 정책 "추정" 배지 필수·합계는 원문 1,000만 고정(비율 배분만 추정). 데이터 출처(약관/
  사규표)는 Human 제공 필요 — 2차 결정 항목.
